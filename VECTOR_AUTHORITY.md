# Vector Authority — Rust Canonical (AC-RC-08 / AC-RC-09)

## Status

**Effective:** 2026-03-13 (RUSTIFY-CORE-1 RC2-EXEC-A)

All golden test vectors in bolt-core-sdk are now generated by the Rust SDK.
TypeScript vector generation scripts are deprecated and retained for
transitional reference only.

## Canonical Vector Locations

| Category | Location | Generator |
|----------|----------|-----------|
| Box-payload | `rust/bolt-core/test-vectors/core/box-payload.vectors.json` | `bolt_core::vectors::generate_box_payload_json` |
| Framing | `rust/bolt-core/test-vectors/core/framing.vectors.json` | `bolt_core::vectors::generate_framing_json` |
| SAS | `rust/bolt-core/test-vectors/core/sas.vectors.json` | `bolt_core::vectors::generate_sas_json` |
| HELLO-open | `rust/bolt-core/test-vectors/core/web-hello-open.vectors.json` | `bolt_core::vectors::generate_hello_open_json` |
| Envelope-open | `rust/bolt-core/test-vectors/core/envelope-open.vectors.json` | `bolt_core::vectors::generate_envelope_open_json` |
| BTR (10 files) | `rust/bolt-core/test-vectors/btr/*.vectors.json` | `bolt_btr::vectors::generate_*_json` |

## Regeneration Commands

```bash
# Regenerate + verify all core vectors (determinism check included)
cargo test -p bolt-core --features vectors --test vector_golden_core

# Regenerate + verify all BTR vectors
cargo test -p bolt-btr --features vectors --test vector_golden

# Run full cross-implementation verification
cargo test -p bolt-core --features vectors  # Rust consumers
cd ts/bolt-core && npm test                  # TS consumers
```

## Deprecated TS Generators (AC-RC-09)

The following scripts are deprecated and will be removed in a future release:

- `ts/bolt-core/scripts/print-test-vectors.mjs` — box-payload + framing
- `ts/bolt-core/scripts/generate-h3-vectors.mjs` — SAS + HELLO-open + envelope-open

These scripts have `@deprecated` JSDoc annotations and emit deprecation
warnings on invocation. Do not use them for new vector generation.

## Roundtrip Verification

The vector pipeline ensures cross-implementation correctness:

1. **Rust generates** → writes to `test-vectors/core/` and `test-vectors/btr/`
2. **Rust consumes** → golden tests verify determinism + decrypt
3. **TS consumes** → same vector files imported and verified
4. **Mismatch → CI failure** in both Rust and TS test suites
